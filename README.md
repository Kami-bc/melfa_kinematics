\# MELFA Kinematics



Repositorio de estudio para organizar y modularizar un notebook académico de cinemática de un robot industrial MELFA de 6 GDL.



Objetivos:

\- aprender Git y GitHub,

\- practicar modularización,

\- usar ambientes virtuales,

\- validar funciones matemáticas,

\- construir una base extensible para cinemática y visualización.
