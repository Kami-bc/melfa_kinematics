\# Caso de estudio MELFA



Este proyecto toma un notebook académico de cinemática de un robot MELFA de 6 GDL y lo convierte gradualmente en un repositorio organizado y extensible.



La meta es aprender:

\- Git

\- GitHub

\- modularización

\- ambientes virtuales

\- validación de funciones

\- estructura básica de proyectos técnicos
